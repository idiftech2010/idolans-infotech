# EduAI One-Page Marketing & Brand Brief

## Product Name
EduAI

## Tagline
AI-powered Learning Management for students, instructors, and institutions.

## Quick Pitch
EduAI is a modern Learning Management System built to help educators deliver courses, manage exams, grade work, and support student success—all with smart AI tools, secure assessment workflows, and role-specific dashboards.

---

## Product Summary
EduAI is a web-based education platform that brings course delivery, assessments, messaging, payments, and AI learning aids together in one polished SaaS experience.

It supports three primary roles:
- Students
- Instructors
- Administrators

The platform powers every stage of the online learning lifecycle:
- Course discovery and enrollment
- Lesson progress and content consumption
- Assignment submission and grading
- Secure exam delivery with AI-assisted proctoring
- AI-powered study tools and content summarization
- Payment processing for paid courses
- Role-specific dashboards and platform analytics

EduAI is designed to brand as a polished, trustworthy, and high-value education product for institutions, educators, and learners.

## Target Audiences

### Students
Students use EduAI to:
- Discover courses by topic, level, and instructor
- Enroll in free or paid classes
- Track course progress and grades
- Submit assignments and receive feedback
- Take AI-proctored exams with integrity protections
- Use built-in AI tools to summarize notes, generate study guides, and build quizzes
- Download completion certificates

### Instructors
Instructors use EduAI to:
- Create and publish courses
- Manage lessons, assignments, and exam content
- Monitor student progress and engagement
- Grade submissions and respond to student messages
- Offer live class experiences and digital collaboration
- Use dashboards to analyze course performance and revenue

### Administrators
Admins use EduAI to:
- Manage users, roles, and access
- Oversee course catalogs, enrollments, and payments
- Track platform usage, active exams, and category performance
- Support compliance, security, and enrollment workflows
- Maintain trust and platform integrity with auditing and reporting

## Core Platform Features

### User Experience
- Responsive one-page landing website with strong conversion messaging
- Secure login, registration, and password recovery flows
- Role-based route protection and dashboard redirection
- Clean modern UI with navigation, cards, progress states, and calls to action

### Course & Learning
- Course browsing with search, category filters, and difficulty levels
- Course detail pages with instructor, category, level, lesson count, enrollment count, and pricing
- Enrollment experience for free and paid courses
- Lesson and course progress tracking with visual progress bars
- Certificate generation and PDF download for completed courses

### Assignments & Grading
- Student assignment submission portal with text-based submissions
- Instructor assignment review and grading workflows
- Assignment status indicators and overdue warnings
- Support for MCQ, essay, and open response assignment types

### Exams & Proctoring
- Exam listing and access from student dashboard
- AI-proctored exam mode with required fullscreen and webcam
- Proctoring detection for tab switching, fullscreen exit, copy/paste attempts, and suspicious activity
- Live webcam feed indicator and integrity score warnings
- Timed exam delivery and automated submission on timeout
- Result display with score, pass/fail, and integrity summary

### AI Learning Tools
- Lecture Summarizer: turn notes, transcripts, and documents into concise summaries
- Note Generator: create structured study notes from pasted text or uploaded files
- Quiz Generator: generate practice questions with options and explanations
- File upload support for PDF, DOCX, PPTX
- Clipboard-friendly result sharing and copy controls

### Messaging & Collaboration
- Messaging inbox for direct communication between users and instructors
- Conversation history with read/unread states
- Start new conversations with another user by ID
- Real-time-style messaging layout, searchable conversation list

### Payments & Monetization
- Demo checkout for free payment preview flows
- Paid checkout support for Stripe and Paystack
- Payment status handling and enrollment confirmation
- Payment history endpoint for auditing and user records
- Payment success routing and transactional UX

### Dashboards & Administration
- Student Dashboard: enrolled courses, upcoming assignments, exam status, messages, grades, and progress metrics
- Instructor Dashboard: active courses, total students, pending reviews, and revenue indicators
- Admin Dashboard: total users, courses, enrollments, active exams, and category analytics
- Admin user management: search and filter by role
- Instructor course creation and management workflows

### Security & Compliance
- JWT authentication with token persistence in local storage
- Role-based access control for pages and API routes
- Backend auth middleware protecting data and operations
- Secure proctoring session logging and suspicious activity scoring

## Technical Architecture

### Frontend
- `artifacts/lms/src`
- React + Vite application
- Wouter router for client-side navigation
- React Query for server state and caching
- Context-based auth flow with JWT token stored in localStorage
- Tailwind CSS styling and reusable UI components

### Backend
- `artifacts/api-server/src`
- Express 5 REST API
- Drizzle ORM for PostgreSQL queries
- Role-based security middleware
- OpenAPI-driven API client generation via `@workspace/api-client-react`

### Data & Services
- PostgreSQL database for users, courses, enrollments, assignments, exams, payments, conversations, and AI session logs
- Stripe and Paystack payment gateway support
- AI endpoints for summarization, notes, and quiz generation
- Auth routes for register, login, forgot password, logout, and user profile

### Deployment & Dev Setup
- Monorepo managed with pnpm workspaces
- Root commands:
  - `pnpm dev`
  - `pnpm dev:frontend-only`
  - `pnpm dev:backend-only`
- Environment variables:
  - `DATABASE_URL`
  - `SESSION_SECRET`
  - `OPENAI_API_KEY`
  - `API_PORT`
  - `VITE_PORT`
  - `BASE_PATH`
- Local development supports launching frontend and backend together
